# Système de recommandation hybride pour e-commerce

## Description

**Projet académique** développé dans le cadre de la Licence en Business Intelligence.  
Ce projet implémente un **moteur de recommandations personnalisées** pour une plateforme e-commerce. Il combine :  

- **Filtrage basé sur le contenu** : analyse des descriptions produits pour recommander des articles similaires.  
- **Filtrage collaboratif** : analyse des évaluations et comportements utilisateurs pour suggérer des produits pertinents.  

L’objectif est de proposer des recommandations adaptées aux préférences individuelles tout en prenant en compte les tendances communautaires.

---

## Fonctionnalités principales

- Prétraitement et vectorisation des descriptions produits (TF-IDF, tokenisation, stemming, suppression des stopwords).  
- Calcul des similarités entre produits et utilisateurs 
- Génération de recommandations personnalisées pour chaque utilisateur.  
- Gestion des évaluations et des achats.  
- Sessions utilisateurs via une application web simple.

---

## Technologies utilisées

- **Langage & Framework** : Python, Flask  
- **Base de données** : MySQL  
- **Bibliothèques Python** : NLTK, NumPy, SciPy, Scikit-learn, Pandas  
- **Frontend** : HTML, CSS, JavaScript  
- **Autres** : PHP (utilisé pour certaines interactions)
